Spark

Spark is an Application Development Framework for creating reliable interactive installations. It has the following features:

  • reliability - based on the Twisted framework, it runs stable and secure. And in the unlikely event that it does crash, it starts itself again.
  • event-based - Twisted is an event based system: everything is executed in a single thread which never blocks on I/O.
  • monitoring - Monitoring your application is built in with support for power connection, network connection, etc.
  • graphics - a graphics system based on the clutter project. Optionally multi-display.
  • much more!

Version: 0.6
